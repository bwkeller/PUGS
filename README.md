# PUGS: The Portable Universal Galaxy Sampler [![build_volume_ic](https://github.com/bwkeller/PUGS/actions/workflows/build_volume_ic.yml/badge.svg)](https://github.com/bwkeller/PUGS/actions/workflows/build_volume_ic.yml) [![linter](https://github.com/bwkeller/PUGS/actions/workflows/linter.yml/badge.svg)](https://github.com/bwkeller/PUGS/actions/workflows/linter.yml)

## `build_volume_ic`
This module contains the code and data required to build the IC for the collisionless volume PUGS uses to sample ICs from.
## `pugs_tangos_db`
This module contains the code to build the tangos database from the ICs generated by `build_volume_ic`, run with your N-body code of choice.
